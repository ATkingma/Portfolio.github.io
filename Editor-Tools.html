I made two tools one that auto saves your unity project to exclude lost progress on an crash and also just for making thinks easier not forgetting to save an scene on project 

the seccond tool I made was an playerpref changer this is ussed to make developers life easier. In use of this tool you can change al set playerprefs instead of setting in the start function the default value or resetting the values.

AutoSaver

I have a float that is the time before it autosaves. You can also manually save. If you want the default save timer to be 5 seconds or 100 then you only need to change "TimeBetweenSaves" and click on the button with "Save Changes" after that it will work. When you press on play it will save the project and your new settings.

I used the following functions to make the tool work.

TimeSinceStartup
SaveScene
SaveAssets
PlaymodeStateChanged
IsPlayingOrWillChangePlaymode
