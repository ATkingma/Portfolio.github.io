This project is all about the curve the axe makes further. I made some basic movements and also a fun combo and attack functions. this project is inspired by god of war what I think is an awesome game with lots of cool feuturs.

Axe Throw

The character moves with an ​character controller and has lots of nice animation features like different layers and an blendtree.

The axe trow used a curve system where it has 3 vector3's and the calculates curve point of the axe then the result of this is an amazing looking curve!


All the ​features,

  • Axe throw with curve
  • Blendtree + animation layers
  • Character controller
  • Multiple attacks : kick,melee,spin
  • Un/equip
  • Taunt
